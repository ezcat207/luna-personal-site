# Todo Star (待办星) - 用户使用指南

## 📖 什么是 Todo Star？

**Todo Star** 是一个基于 SOP（标准操作流程）的时间管理和任务管理工具，帮助你从被动应对转向主动管理，提升每日工作效率。

---

## 🎯 核心概念：8 个 SOP 工作流

### 当前可用（Phase 1 MVP）：

#### 1. **今日计划 SOP** (`/todo/today`)
**目标**：让你的一天从"主动模式"开始

**如何使用**：
1. **早上第一件事**：打开 `/todo/today` 页面
2. **快速添加任务**：
   - 在底部输入框输入任务标题
   - 按 `Enter` 提交
   - 按 `Esc` 清空输入
3. **标记 Top 3 优先级**：
   - 点击任务右侧的 ⭐ 星标图标
   - 最多只能标记 3 个最重要的任务
   - Top 3 任务会显示在页面顶部的大卡片中
4. **开始执行**：
   - 从 Top 3 中选择第一个任务开始
   - 完成后勾选复选框 ✓
5. **持续更新**：
   - 完成一个任务后，立即添加下一个
   - 随时调整 Top 3（移除完成的，添加新的重点）

**关键原则**：
- ✅ **每天只专注 3 件最重要的事**（不是 10 件，不是 20 件）
- ✅ **优先级 > 数量**（完成 3 件重要事 > 完成 30 件琐事）
- ✅ **动态调整**（计划赶不上变化，随时重新标记 Top 3）

---

#### 2. **番茄专注 SOP** (`/todo/focus`) - 即将上线
**目标**：用 25 分钟的专注块对抗拖延和分心

**预期使用流程**：
1. 从今日计划中选择一个任务
2. 点击 "🍅 Start Pomodoro" 进入全屏专注模式
3. 设定 25 分钟倒计时，开始工作
4. 期间不查看手机、不回消息、不切换任务
5. 25 分钟到后，休息 5 分钟
6. 完成 4 个番茄钟后，休息 15-30 分钟

**关键原则**：
- ✅ **一次只做一件事**（不要多任务并行）
- ✅ **25 分钟不中断**（真正的深度工作）
- ✅ **完成即胜利**（不求完美，先完成一版）

---

#### 3. **每日复盘 SOP** (`/todo/review`) - 即将上线
**目标**：每天进步一点点，避免重复犯错

**预期使用流程**（每天收尾 10 分钟）：
1. **查看今天完成了什么**（自动显示已完成的任务）
2. **回答 3 个问题**：
   - 💭 今天遇到的一个问题：例如"被 Slack 通知打断，无法专注"
   - 💡 明天会做的一个改进：例如"上午关闭 Slack 通知"
   - 🌅 明天第一件事：例如"继续完成项目提案"
3. **保存复盘**（自动记录到数据库）

**关键原则**：
- ✅ **不盯着没做完的事**（先肯定已完成的成果）
- ✅ **每天只改进一件事**（不要列 10 个改进点）
- ✅ **具体 > 模糊**（"明天 10 点前完成 PPT" > "明天努力工作"）

## 🚀 快速开始（3 步上手）

### Step 1: 登录账号
1. 访问 `log.bunnyuniverse.com/todo/today`
2. 点击右上角 "Sign In" 按钮
3. 使用 Google 账号登录（数据会自动同步到云端）

### Step 2: 添加第一个任务
1. 在页面底部看到输入框："What needs to be done today?"
2. 输入任务标题，例如：`写项目提案初稿`
3. 按 `Enter` 提交

### Step 3: 标记 Top 3
1. 点击任务右侧的 ⭐ 星标图标
2. 任务会移动到页面顶部的 "Top 3 Priorities" 区域
3. 重复操作，标记今天最重要的 3 件事

## 🎨 界面元素说明

### 任务卡片（Task Card）

```
┌─────────────────────────────────────────┐
│ [ ] 写项目提案初稿            [⭐] [×]  │
│     Due: Tomorrow                        │
│     [🔴 High]                            │
└─────────────────────────────────────────┘

[ ] = 完成复选框（点击标记完成）
⭐ = 添加到 Top 3（最多 3 个）
× = 删除任务
🔴 High / 🟡 Med / 🟢 Low = 优先级徽章
```

### Top 3 优先级卡片（Hero Card）

```
┌────────────────────────────────────────────┐
│  ⭐ 1. [ ] 修复线上 bug                    │
│          10:00 - 11:30                     │
│                                            │
│  ⭐ 2. [ ] 写周报                          │
│          14:00 - 15:30                     │
│                                            │
│  ⭐ 3. [ ] 准备下午会议材料                │
│          16:00 - 17:00                     │
└────────────────────────────────────────────┘

渐变紫色背景 = Top 3 专属视觉强调
时间段显示 = 可选（点击任务编辑添加）
```

## ⌨️ 键盘快捷键

- `Enter` - 提交任务（在输入框中）
- `Esc` - 清空输入框
- 点击任务标题 - 进入编辑模式

## 🔄 数据同步说明

- ✅ **自动保存**：所有操作立即保存到云端（Supabase）
- ✅ **跨设备同步**：在电脑上添加任务，手机上也能看到
- ✅ **实时更新**：打开多个标签页，数据自动同步

## 🛡️ 隐私和数据安全

- ✅ **数据隔离**：每个用户只能看到自己的任务（RLS 策略）
- ✅ **安全认证**：使用 Google OAuth 登录（Supabase Auth）
- ✅ **数据加密**：传输加密（HTTPS）+ 存储加密（Supabase）

## 🆘 常见问题

### Q1: 为什么只能标记 3 个 Top 3？
**A**: 这是刻意设计的限制。研究表明，人一天真正能专注完成的重要任务不超过 3 件。强制限制帮助你做出真正的优先级选择。

### Q2: 任务可以设置截止日期吗？
**A**: MVP 版本暂不支持。你可以在任务标题中手动添加日期，例如：`写周报 (Due: 6/30)`

### Q3: 可以创建子任务吗？
**A**: MVP 版本暂不支持。建议把大任务拆分成多个独立任务。

### Q4: 数据会丢失吗？
**A**: 不会。所有数据存储在 Supabase 云端，只要你登录同一个 Google 账号，数据永久保留。

### Q5: 可以导出任务列表吗？
**A**: 暂不支持（计划在 Phase 2 添加 CSV 导出功能）
